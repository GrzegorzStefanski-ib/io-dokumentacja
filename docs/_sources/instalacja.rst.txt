Instalacja
===============

Wymagania
----------------

- Docker
- Docker Compose
- Java 8
- PostgreSQL


Budowa i uruchomienie kontenera
--------------------------------------

Aby zbudować obraz należy posłużyć się komendą

.. code-block:: RST

   docker-compose build


By uruchomić aplikacją należy posłużyć się komendą

.. code-block:: RST

   docker-compose up
